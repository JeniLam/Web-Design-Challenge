Summary: Latitude vs. X

The purpose of the WeatherPy project was to create a Python script to visualize the weather of at least 500 cities across the world of varying distance from the equator. To accomplish this, a simply Python library was used along the with OpenWeatherMap API.

After gathering the data necessary, it was required to create a series of scatter plots to showcase the relationship, if any, between latitude and temperature, humidity, cloudiness and wind speed. We were also required to then run linear regressions on each relationship after seperating the datasets into Northern Hemisphere (greather than or equal to 0 degrees latitude) and Southern Hemispheres (less than 0 degrees latitude).
